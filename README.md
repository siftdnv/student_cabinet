# student_cabinet
Прототип вебсайта личного кабинета студента. Включает различные приложения, такие как расписание, которое интегрирует реальное с ИСУ, зачетная книжка, учебные курсы, успеваемость. Также реализована авторизация/регистрация. Создана панель настроек аккаунта / внешнего вида приложения.
